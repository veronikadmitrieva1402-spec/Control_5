# Генератор случайных цитат

## Подготовка проекта

1. Скачайте архив проекта или склонируйте репозиторий
2. Распакуйте архив в нужную папку
3. Откройте терминал/командную строку в папке проекта

## Перейдите в папку проекта
 - cd путь_к_папке_проекта

## Установите зависимости
 - npm install
 
## Запуск сервера
 - npm run dev

## Проверка работы
1. Откройте браузер и перейдите по адресу: http://localhost:3000
